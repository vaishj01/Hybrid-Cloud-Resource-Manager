# Hybrid Cloud Resource Manager

## Project Overview

Hybrid Cloud Resource Manager is a full-stack cloud management platform that enables users to provision, monitor, and manage AWS cloud resources through a secure web interface.

## Features

- User Authentication
- Dashboard
- AWS EC2 Management
- AWS S3 Management
- Infrastructure Automation using Terraform
- Cloud Monitoring
- Deployment History
- Docker Support
- CI/CD using GitHub Actions
